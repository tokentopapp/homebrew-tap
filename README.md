# homebrew-tap

Homebrew formulae for [tokentop](https://github.com/tokentopapp/tokentop).

## Install

```bash
brew install tokentopapp/tap/tokentop
```

## Update

```bash
brew upgrade tokentop
```

## About

This tap is automatically updated by the [tokentop release workflow](https://github.com/tokentopapp/tokentop/blob/main/.github/workflows/release.yml) on each new release. SHA256 hashes are computed from the release artifacts and committed here by CI.
